HTML

  • The head element contains meta data, language & charset info.
  • The Doctype html declaration is used to let the browser know the language used
  • The body element represents the visible content shown to the user.
  • Angle brackets are used to open and close tags
  • The HTML elements structure the page and act as building blocks.
  • The h1 to h6 tags are tiered heading tags.
  • Tags help to create elements and their child elements
  • The P tag can be used to define a paragraph or line's of text.
  • Class and id attributes can be used in html markup as selectors for styling in css

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Css is cascading style sheets.
  • Css can be styled inline or in the header of html using style tags.
  • Linking a seperate stylesheet to your html doc is the preferred method for developers.
  • Elements can be positioned and styled with CSS.
  • Class and id attributes can be used in html markup as selectors for styling in CSS.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Git push is how you transfer commits from your local repository to a remote repository main branch
  • Git add -A moves files to staging area.
  • Create new files using 'touch' keyword
  • Git pull request to discuss and review the potential changes before merge
  • Cd (your directory) and Cd .. to navigate back into directory in gitbash cli

JavaScript

  • Console.log can be use in JavaScript to log ouputs to the console
  • Loops and conditional statements can be used to iterate thru arrays
  • Boolean true and false statements to define if a value is equal to ===
  • JavaScript needs to be linked to the html footer and body closing tags using script.js
  • Var, let and const are keywords used to assign variables
  • Chrome dev tools can be used to see the result of javascript code in action
