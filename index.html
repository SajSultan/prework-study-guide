HTML

  • The head element contains information about the webpage such as meta data, language and unix character set info.
  • The Doctype html declaration is used to let the browser know the language used
  • The body element represents the visible content shown to the user.
  • Angle brackets are used to open and close tags
  • The HTML elements structure the page and act as building blocks.
  • The h1 to h6 tags are tiered heading tags.
  • Tags help to create elements and their child elements
  • The P tag can be used to define a paragraph or line's of text.
  • Class and id attributes can be used in html markup as selectors for styling in css
